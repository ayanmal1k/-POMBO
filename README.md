## 📖 The Lore

Pavel Durov loves pigeons — he loves what they represent.

Before the paper plane, there were pigeons. Telegram is the modern messenger. Pigeons were the **original messengers**: autonomous, resilient, and built to deliver messages across distance.

That is why **POMBO** fits.

$POMBO brings in one of the most recognizable pigeon-like birds on the internet: the **deadpan WhatsApp bird**. Officially, Unicode calls it *Bird*. Visually, everyone reads it as a weird pigeon. In Portuguese, ***Pombo*** literally means pigeon.

But $POMBO is not only about the messenger analogy. That is just the foundation. The real edge is that Pombo already works as **internet culture**.

> **Pombo is the face of:**
> *"I don't know what is happening, but I'm witnessing it."*

- Funny but serious.
- Serious but stupid.
- Sarcasm with a straight face.

Telegram never had a famous pigeon. WhatsApp accidentally did. So Pombo picked a side.

## 🎨 The Art

Pombo is **not** a random bird drawing.

The WhatsApp emoji set was created by [**The Iconfactory**](https://design.iconfactory.com/), a legendary design studio that helped shape digital visual culture for decades. They worked on emoji sets for WhatsApp, Twitter and Facebook Messenger, created Twitterrific — one of the most influential early Twitter clients — and even produced visual/icon work for Windows XP, one of the most recognizable operating systems in internet history.

## 🌐 The Website

A premium, fully animated Next.js site built with:

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **GSAP** (ScrollTrigger, Flip) for scroll-driven animations
- **Framer Motion** for layout animations
- **Responsive** design — mobile-first
